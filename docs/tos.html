ข้อกำหนดการให้บริการ

PW Discord Bot

เงื่อนไขการให้บริการสำหรับบอท PW Discord Bot ซึ่งรวมถึงการเก็บข้อมูล การใช้งาน และบริการภายนอกที่เชื่อมต่อ

บอท PW Discord Bot

discordbot.pwisetthon.com

1. ข้อมูลทั่วไป

บอท PW Discord Bot ("บอท") ทำงานบนแพลตฟอร์ม Discord การเชิญหรือใช้งานบอทนี้ในเซิร์ฟเวอร์ Discord ของคุณ ถือว่าคุณยอมรับข้อกำหนดการให้บริการฉบับนี้ทั้งหมด

2. ขอบเขตการให้บริการ

บอทมีฟีเจอร์และคำสั่งดังต่อไปนี้:

2.1 บริการสลากกินแบ่งรัฐบาล

  • /lastlotto — ดูผลสลากกินแบ่งรัฐบาลงวดล่าสุด
  • /subthlotto — ติดตาม/ยกเลิกการแจ้งเตือนผลสลากฯ อัตโนมัติ
  • /flottomode — ปรับโหมดการแสดงผลสลากฯ (ปกติ / โกลด์)
  • /lotsheet — แสดงใบตรวจสลากกินแบ่งรัฐบาล
  • /synumber — บันทึกเลขสลากฯ ที่คุณซื้อเพื่อรับการแจ้งเตือนผลรางวัล
  • /srchlot — ตรวจสลากฯ ล่าสุดด้วยเลขที่กำหนด
  • /ตรวจสลากฯ — คำสั่งตรวจสลากฯ แบบ Context Menu
  • /syhistory — ดูประวัติการบันทึกเลขสลากฯ ของคุณ

บอทจะแจ้งเตือนผลสลากฯ อัตโนมัติในช่วงเวลา 15:00–17:00 น. ของทุกวันที่มีการออกสลาก โดยอ้างอิงข้อมูลจาก rapidapi.com/boyphongsakorn/api/thai-lottery1 และ สำนักงานสลากกินแบ่งรัฐบาล (glo.or.th)

2.2 บริการราคาน้ำมัน

  • /lastthaioilprice — ดูราคาน้ำมันล่าสุดในประเทศไทย
  • /subthaioilprice — ติดตาม/ยกเลิกการแจ้งเตือนเมื่อราคาน้ำมันเปลี่ยนแปลง

บอทดึงข้อมูลราคาน้ำมันจาก thaioilpriceapi-vercel.vercel.app และจะแจ้งเตือนช่องที่ติดตามโดยอัตโนมัติเมื่อมีราคาใหม่

2.3 บริการอื่น ๆ

  • /checkconnection — ตรวจสอบสถานะการเชื่อมต่อของบอท

3. การเก็บข้อมูลส่วนบุคคล

3.1 ข้อมูลที่บอทจัดเก็บ

  • Discord User ID — เพื่อส่งแจ้งเตือนผลสลากฯ ผ่านข้อความส่วนตัว (DM)
  • Discord Channel ID — เพื่อส่งผลสลากฯ และราคาน้ำมันเข้าช่องที่ลงทะเบียน
  • Discord Guild (Server) ID — เพื่อบริหารจัดการการตั้งค่าต่าง ๆ ต่อเซิร์ฟเวอร์
  • เลขสลากฯ ที่ผู้ใช้บันทึก — เก็บในฐานข้อมูลเพื่อใช้ตรวจผลรางวัลอัตโนมัติ

3.2 วิธีการจัดเก็บข้อมูล

ข้อมูลทั้งหมดจัดเก็บในฐานข้อมูล MySQL ที่ผู้พัฒนาดูแลเอง ข้อมูลไม่ถูกส่งต่อให้บุคคลที่สาม ยกเว้นที่จำเป็นเพื่อการให้บริการ (เช่น การดึง API ราคาน้ำมันหรือผลสลากฯ)

3.3 การใช้งานข้อมูล

ข้อมูลที่เก็บรวบรวมจะถูกใช้เพียงเพื่อวัตถุประสงค์ของการให้บริการบอทเท่านั้น ไม่มีการนำไปใช้เพื่อการตลาด การโฆษณา หรือการขายข้อมูล

4. ข้อจำกัดความรับผิดชอบ

4.1 ความถูกต้องของข้อมูล

  • ผลสลากกินแบ่งรัฐบาลที่แสดงอ้างอิงจาก API ภายนอก หากข้อมูลผิดพลาดกรุณาตรวจสอบจาก glo.or.th โดยตรง
  • ราคาน้ำมันที่แสดงอาจมีความล่าช้าหรือคลาดเคลื่อนจากราคาจริง กรุณาตรวจสอบจากผู้จำหน่ายน้ำมันโดยตรง

4.2 ความพร้อมในการให้บริการ

บอทให้บริการในลักษณะ "ตามที่มี" (as-is) ผู้พัฒนาไม่รับประกันว่าบอทจะพร้อมใช้งานตลอดเวลา อาจมีการหยุดให้บริการชั่วคราวเพื่อการบำรุงรักษา อัปเดต หรือเหตุขัดข้องทางเทคนิค

4.3 การตัดสินใจของผู้ใช้

ผู้ใช้ต้องใช้วิจารณญาณของตนเองในการตัดสินใจโดยอาศัยข้อมูลจากบอท โดยเฉพาะในเรื่องการซื้อสลากฯ และราคาน้ำมัน ผู้พัฒนาไม่รับผิดชอบต่อความเสียหายใด ๆ ที่เกิดจากการใช้ข้อมูลจากบอท

5. พฤติกรรมที่ห้ามกระทำ

ผู้ใช้งานและผู้ดูแลเซิร์ฟเวอร์ต้องไม่:

  • พยายามล่วงละเมิด หรือทดสอบช่องโหว่ของระบบบอท
  • ส่งคำสั่งในปริมาณมากเกินสมควรจนกระทบต่อการให้บริการ (Abuse / Spam)
  • นำข้อมูลที่ได้จากบอทไปใช้ในทางที่ผิดกฎหมาย
  • พยายามหลอกลวงหรือแอบอ้างว่าตนเองเป็นผู้พัฒนาบอท

6. API และบริการภายนอกที่ใช้

บอทนี้เชื่อมต่อกับบริการภายนอกดังต่อไปนี้ ซึ่งมีข้อกำหนดการให้บริการของตนเอง:

  • สำนักงานสลากกินแบ่งรัฐบาล (glo.or.th)
  • RapidAPI – Thai Lottery API (rapidapi.com)
  • Thai Oil Price API (thaioilpriceapi-vercel.vercel.app)
  • Discord API (discord.com)

ผู้ใช้ยอมรับว่าการใช้งานบอทหมายถึงการยินยอมให้บอทดึงข้อมูลจากบริการเหล่านี้ในนามของการให้บริการ

7. การยุติการใช้งาน

7.1 โดยผู้ใช้

ผู้ดูแลเซิร์ฟเวอร์สามารถถอนบอทออกจากเซิร์ฟเวอร์ได้ทุกเมื่อ และสามารถยกเลิกการแจ้งเตือนต่าง ๆ ผ่านคำสั่ง /subthlotto และ /subthaioilprice

7.2 โดยผู้พัฒนา

ผู้พัฒนาสงวนสิทธิ์ในการระงับหรือยุติการให้บริการแก่เซิร์ฟเวอร์หรือผู้ใช้ที่ละเมิดข้อกำหนดนี้ โดยไม่จำเป็นต้องแจ้งล่วงหน้า

8. การแก้ไขข้อกำหนด

ผู้พัฒนาอาจแก้ไขข้อกำหนดการให้บริการนี้ได้ตลอดเวลา การใช้งานบอทต่อเนื่องหลังจากมีการแก้ไขถือว่าคุณยอมรับข้อกำหนดฉบับใหม่
