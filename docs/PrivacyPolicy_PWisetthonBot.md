## 1. บทนำ

TeamQuadB.in.th ("ผู้พัฒนา") ให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้งานบอท PWisetthon Discord Bot ("บอท") เป็นอย่างยิ่ง นโยบายนี้อธิบายว่าบอทเก็บข้อมูลอะไร เก็บอย่างไร ใช้เพื่ออะไร และมีสิทธิ์อะไรบ้าง

การเชิญหรือใช้งานบอทในเซิร์ฟเวอร์ Discord ของคุณถือว่าคุณรับทราบและยอมรับนโยบายฉบับนี้

---

## 2. ข้อมูลที่บอทเก็บรวบรวม

### 2.1 ข้อมูลที่เก็บโดยอัตโนมัติเมื่อเชิญบอท

เมื่อบอทถูกเชิญเข้าเซิร์ฟเวอร์ ระบบจะจัดเก็บข้อมูลต่อไปนี้โดยอัตโนมัติ:

| ข้อมูล | วัตถุประสงค์ |
|---|---|
| Discord Server (Guild) ID | ระบุตัวตนเซิร์ฟเวอร์เพื่อบริหารการตั้งค่า |
| Discord Channel ID (System Channel) | ส่งข้อความต้อนรับและแจ้งเตือนสลากฯ เริ่มต้น |
| ชื่อเซิร์ฟเวอร์ | บันทึก Log เพื่อการดูแลระบบ |

### 2.2 ข้อมูลที่เก็บเมื่อผู้ใช้ลงทะเบียนการแจ้งเตือน

เมื่อผู้ใช้หรือผู้ดูแลเซิร์ฟเวอร์ใช้คำสั่งติดตามบริการต่าง ๆ:

| ข้อมูล | คำสั่งที่เกี่ยวข้อง | วัตถุประสงค์ |
|---|---|---|
| Discord Channel ID | `/subthlotto`, `/subthaioilprice` | ส่งแจ้งเตือนสลากฯ และราคาน้ำมันเข้าช่องที่เลือก |
| Discord Guild ID | `/subthlotto`, `/flottomode` | บันทึกการตั้งค่าโหมดการแสดงผลต่อเซิร์ฟเวอร์ |

### 2.3 ข้อมูลที่เก็บเมื่อผู้ใช้บันทึกเลขสลากฯ

เมื่อผู้ใช้ใช้คำสั่ง `/synumber`:

| ข้อมูล | วัตถุประสงค์ |
|---|---|
| Discord User ID | ระบุตัวตนเพื่อส่งผลแจ้งเตือนกลับผ่าน DM |
| ตัวเลขสลากฯ ที่บันทึก | ใช้ตรวจสอบกับผลรางวัลและแจ้งเตือนอัตโนมัติ |
| สถานะของเลข (waiting / win / ไม่ถูก) | ติดตามผลการตรวจรางวัล |
| งวดสลากฯ ที่ตรวจ | บันทึกประวัติการตรวจ |

### 2.4 ข้อมูลที่ไม่ได้เก็บ

บอทไม่เก็บข้อมูลดังต่อไปนี้:
- เนื้อหาข้อความในช่องแชท (บอทไม่อ่านข้อความทั่วไป)
- ข้อมูลส่วนตัวอื่น ๆ เช่น ชื่อจริง อีเมล หรือเบอร์โทรศัพท์ของผู้ใช้
- ข้อมูลทางการเงินของผู้ใช้

## 3. วิธีการจัดเก็บข้อมูล

### 3.1 ฐานข้อมูล
ข้อมูลทั้งหมดจัดเก็บในฐานข้อมูล MySQL ที่ผู้พัฒนาบริหารจัดการเอง บนเซิร์ฟเวอร์ส่วนตัว ไม่ได้ใช้บริการคลาวด์สาธารณะในการจัดเก็บข้อมูลผู้ใช้

### 3.2 ระยะเวลาการเก็บข้อมูล
- **ข้อมูลช่องแจ้งเตือน** — เก็บตราบเท่าที่ยังไม่มีการยกเลิกการติดตาม
- **เลขสลากฯ ที่บันทึก** — เก็บจนกว่าจะมีการตรวจผลแล้วเสร็จ (เปลี่ยนสถานะจาก "waiting" เป็น "win" หรือ "ไม่ถูก")
- **ข้อมูล Log การทำงาน** — เก็บชั่วคราวเพื่อการดูแลระบบ

### 3.3 การรักษาความปลอดภัย
- รหัสผ่านฐานข้อมูลและ Token ของบอทเก็บเป็น Environment Variable ไม่ได้ฝังในโค้ดโดยตรง
- การเชื่อมต่อฐานข้อมูลใช้ Connection Pool พร้อมระบบ Reconnect อัตโนมัติ

## 4. วิธีการใช้ข้อมูล

ข้อมูลที่เก็บรวบรวมถูกนำไปใช้เพื่อวัตถุประสงค์ดังต่อไปนี้เท่านั้น:

1. **ส่งแจ้งเตือนผลสลากกินแบ่งรัฐบาล** — แจ้งเตือนช่องที่ลงทะเบียนและผู้ใช้ที่บันทึกเลขสลากฯ
2. **ส่งแจ้งเตือนราคาน้ำมัน** — แจ้งเตือนช่องที่ลงทะเบียนเมื่อราคาน้ำมันเปลี่ยนแปลง
3. **ตอบสนองต่อคำสั่ง** — ประมวลผลและตอบกลับคำสั่ง Slash Command ของผู้ใช้
4. **ดูแลระบบ** — ติดตามสถานะการทำงานและแก้ไขปัญหา

ผู้พัฒนาไม่นำข้อมูลไปใช้เพื่อการโฆษณา การตลาด หรือขายให้บุคคลที่สาม

## 5. การแบ่งปันข้อมูลกับบุคคลที่สาม

### 5.1 API ที่บอทเรียกใช้
บอทส่งข้อมูลบางส่วนไปยังบริการภายนอกเพื่อให้บริการได้ครบถ้วน:

| บริการ | ข้อมูลที่ส่งไป | วัตถุประสงค์ |
|---|---|---|
| lotapi.pwisetthon.com | วันที่ + เลขสลากฯ | ตรวจผลรางวัลสลากฯ |
| thaioilpriceapi-vercel.vercel.app | (ไม่มีข้อมูลส่วนบุคคล) | ดึงราคาน้ำมัน |
| whoscheat.com | ชื่อ / เลขบัญชี / เบอร์ที่ผู้ใช้ค้นหา | ตรวจสอบประวัติการโกง |
| chaladohn.com | ชื่อ / เลขบัญชี / เบอร์ที่ผู้ใช้ค้นหา | ตรวจสอบประวัติการโกง |
| Discord API | ข้อมูลที่จำเป็นสำหรับการส่งข้อความ | ส่งผลลัพธ์กลับผู้ใช้ |

### 5.2 สิ่งที่ไม่ได้ทำ
- ไม่ขายข้อมูลผู้ใช้
- ไม่แบ่งปันข้อมูลกับบุคคลที่สามนอกเหนือจากที่ระบุข้างต้น
- ไม่ใช้ข้อมูลผู้ใช้เพื่อวัตถุประสงค์อื่นนอกจากการให้บริการ

## 6. สิทธิ์ของผู้ใช้

คุณมีสิทธิ์ดังต่อไปนี้เกี่ยวกับข้อมูลของคุณ:

**ยกเลิกการแจ้งเตือน**
ใช้คำสั่ง `/subthlotto` หรือ `/subthaioilprice` เพื่อยกเลิกการแจ้งเตือนและลบ Channel ID ของคุณออกจากระบบ

**ถอนบอทออกจากเซิร์ฟเวอร์**
ผู้ดูแลเซิร์ฟเวอร์สามารถ Kick บอทออกได้ทุกเมื่อ ข้อมูลของเซิร์ฟเวอร์นั้นจะหยุดถูกประมวลผล

**ขอลบข้อมูล**
หากต้องการลบข้อมูลของคุณออกจากระบบทั้งหมด กรุณาติดต่อผู้พัฒนาผ่านช่องทางที่ระบุในหมวด 9

## 7. การใช้งาน Discord API

บอทนี้ใช้ Discord API โดยมีสิทธิ์ (Intents) ดังต่อไปนี้:
- **Guilds** — เพื่ออ่านข้อมูลเซิร์ฟเวอร์และช่อง
- **Guild Messages** — เพื่อรับและตอบสนองต่อคำสั่ง Slash Command

บอทไม่มีสิทธิ์อ่านข้อความทั่วไปในช่อง (ไม่ได้ใช้ Message Content Intent)
